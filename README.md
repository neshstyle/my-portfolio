## 👋 Overview

A dark luxury tech portfolio built from the ground up over the Winter 2026 semester at Humber College as part of the Computer Programming & Analysis program (CPAN-134 Web/HTML & CSS). What started as a basic HTML page evolved across 4 chapters into a fully interactive, animated, and deployed web application — complete with matrix rain, particle explosions, and flip cards. 🔥

---

## ✨ Features

### 📐 Chapter 1 — HTML Structure & Version Control
- Semantic HTML5 structure with header, hero, about, projects, contact, and footer sections
- Git repository initialized and pushed to GitHub
- Deployed live via GitHub Pages 🚀

### 🎨 Chapter 2 — CSS Styling & Responsive Design
- Full custom CSS with dark luxury theme — black, navy, gold, teal, and orange accent system ✨
- Typography: Playfair Display (display) + JetBrains Mono (mono)
- Sticky header with gold hover underline animations
- Fully responsive — mobile, tablet, and desktop breakpoints 📱💻🖥️
- Fixed footer

### ⚡ Chapter 3 — JavaScript Interactivity
- 🖥️ **Loading Screen** — Terminal-style boot sequence with matrix rain background, typing animation, and gold progress bar
- ✨ **Particle Constellation** — Interactive gold particle network in the hero section with mouse repulsion and click-to-explode effect (Canvas API)
- 📊 **Scroll Progress Bar** — Gold/orange gradient progress indicator at the top of the page
- 🎬 **GSAP Animations** — Smooth scroll-triggered entrance animations on all sections (external library)
- 🃏 **Vertical Flip Cards** — Project cards flip on hover (desktop) and tap (mobile) to reveal details
- 🧲 **Magnetic Hover** — Project cards subtly pull toward the cursor
- 📬 **Contact Form Validation** — JS validation on name, email, and message fields with error states and success toast notification
- 🔢 **Matrix Rain** — Animated Japanese katakana falling code background in the contact section (Canvas API)

### 🔧 Chapter 4 — Continuous Improvement
- 🔍 **Project Filtering** — Category filter buttons (All / Film / Events / AI / Web Dev) show/hide cards dynamically
- 🛠️ **Tech Stack Skills Section** — Visual skill tags added to the About section
- 🔆 **Active Nav Highlight** — Navigation links highlight gold as you scroll through sections
- ⚡ **Lazy Loading** — Images load on demand for better performance
- 📱 **StudySync Project Card** — Added real group project with live link
- 🔗 **Live Project Links** — Direct links to deployed projects on card backs
- 📝 **Content Updates** — About section updated with current tech stack and graduation year
